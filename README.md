A simple and interactive **task management board** built using **JavaScript, HTML, and CSS**, with drag & drop functionality and status tracking.

---

## **Features**

- **Dynamic Team Setup:** Add multiple team members via a SweetAlert2 popup.
- **Drag & Drop Tasks:** Move tasks between team member columns effortlessly.
- **Task Status:** Mark tasks as *Not Selected*, *On Going*, or *Finished* with a dropdown.
- **Real-Time Task Count:** Each column displays the number of tasks dynamically.
- **Delete Tasks:** Remove tasks easily with a single click.
- **Responsive & Interactive UI:** Hover animations, colored task statuses, and clean layout.

---

## **How to Use**

1. Open `index.html` in your browser.
2. Enter your team member names in the popup (separated by commas).
3. Add tasks using the input box at the left column.
4. Drag tasks to different members and update their status as needed.
5. Task counts update automatically.
